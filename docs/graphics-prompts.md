# Graphics Prompts

Use these prompts in ChatGPT image generation if you want to replace the code-native visuals with richer bitmap artwork.

## Dashboard Page Background

```text
Create a wide, calm background illustration for a personal running tracker dashboard. Show a hobby runner on a quiet Czech countryside road at sunrise, with soft rolling hills, a winding route line, light morning mist, and plenty of negative space on the left for dashboard text and cards. Style: premium product illustration, warm and friendly but restrained, not cartoonish, no text, no logos, no medical claims, landscape 21:9.
```

## Race Event Card

```text
Create a clean race preparation poster illustration for a personal running app. Show a race bib, road curve, soft Czech town or forest edge, and subtle distance markers. Style: modern editorial sports poster, warm but restrained colors, no text, no brand names, no faces, portrait 4:5.
```

## Empty State

```text
Create a friendly empty-state illustration for a running training app. Show unlaced running shoes next to a simple route map line and morning light. Style: clean product illustration, simple shapes, generous whitespace, no text, no logos, transparent-looking light background, square.
```

## Heatmap

```text
Create a visual concept for a running route heatmap. Show an abstract map with layered route traces and warmer hotspots where runs overlap, with a Czech urban park and trail feeling. Style: modern app illustration, clear route density, no real map labels, no text, no logos, landscape 16:9.
```
